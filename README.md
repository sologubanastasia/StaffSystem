StaffSystem — система обліку працівників компанії

Тестове завдання: створення ASP.NET Core застосунку для обліку персоналу з використанням PostgreSQL і Docker.
Проєкт реалізує систему обліку працівників з такими даними: відділ, посада, ПІБ, адреса, телефон, дата народження, дата взяття на роботу, оклад, інформація про компанію.
Дані зберігаються в PostgreSQL і структуровані відповідно до нормалізації. Логіка розділена на рівні Application, Domain, Infrastructure, Web.

Меню додатку включає головну сторінку з інформацією про компанію, сторінку перегляду персоналу з фільтрами для швидкого пошуку, картку працівника з можливістю редагування, 
зарплатну звітність з фільтрами і сумами окладів, а також можливість вивантаження відфільтрованих даних у TXT-файл.

Використані технології: C# / ASP.NET Core 8.0, PostgreSQL, Entity Framework Core, Docker.

База даних PostgreSQL, запускається у контейнері Docker.

Інструкція для запуску:

Клонувати репозиторій:
git clone https://github.com/sologubanastasia/StaffSystem.git

Перейти в папку з веб-проєктом:
cd StaffSystem/StaffSystem.Web

Запустити Docker-композицію:
docker-compose up --build

Це підніме контейнер з PostgreSQL, побудує і запустить веб-додаток, автоматично застосує міграції до бази.
Після запуску застосунок доступний у браузері за адресою: http://localhost:8080/

Docker-compose використовує образ PostgreSQL з базою StaffSystem і веб-додаток StaffSystem.Web. Порти: веб-додаток 8080, PostgreSQL 5432.
